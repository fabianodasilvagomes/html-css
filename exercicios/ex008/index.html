Principais formatações

Negrito / Destaque

para deixar em negrito pode ser utlisado a tag B, (não semântica ) por isso não muito recomendada.

tambem pode ser utilizado a tag strong para realizar esse destaque,(tag semântica)

Italico / Ênfases

Para deixar o texto em italico pode ser utilizado a tag i, (Não semântica), por isso não muito recomendada

tambem pode ser utilizado a tag em, (tag semântica )

Para fazer formatações de textos em HTML5 é recomendado que faça de forma mais semântica possivel, para dar sentido as coisas que esta fazendo e não deixar seu textos bagunçados, tem formas que pode ser mais praticas, mais não quer dizer que esta certa, tome cuidado com isso.

Sempre praticar para não esquecer os conceitos da línguagem

Marcação de texto

Podemos marcar uma parte do texto utilizando a tag MARK.

Texto pequeno

Posso deixar o texto um ©pouquinho menor utilizando a tag SMALL.

Texto deletado

Eu posso marcar um texto pra falar que ele foi excluido para indicar que o texto deve ser lido mais não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma enfâses nele e indicar que foi inserido depois

Texto sobrescrito

Para adiconar coisas do tipo x20 + 3

Texto subscrito

Para adiconar coisas do tipo H2O
